## What is git?
A: Free and open source version control system.

What is version control?
A: The management of changes to documents, computer programs, large websites, and other collections of information.

Terms:
Directory -> Folder 
Terminal or Command Line -> Interface for text commands
CLI -> Command Line Interface
cd -> change directory
Code Editor -> Word processor for writing code
Repository -> Project, or the folder/place where your project is kept
git -> A tool to track code changes
github  -> A website to host your repositories online 


## git commands

add -> track files and changes in git
commit -> save files in git
